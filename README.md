# NexaClaro Website

Премиум веб-страница за NexaClaro — модерни веб-страници, бизнис платформи и дигитални решенија.

## Технологии

- Vite + React + TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Локално стартување

```bash
npm install
npm run dev
```

Отворете [http://localhost:5173](http://localhost:5173)

## Production build

```bash
npm run build
npm run preview
```

## Ассети

- `public/assets/logo.png` — лого
- `public/assets/platform-demo.webm` — демо видео на платформата
